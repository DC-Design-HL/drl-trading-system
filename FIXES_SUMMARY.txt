TECHNICAL NOTES

RegimeInfo dataclass structure:
- trend_strength: ADX value (0-100)
- trend_direction: +1 bullish, -1 bearish, 0 neutral
- volatility_ratio: Current ATR / Average ATR

Order Flow Enhanced Signal Structure (3 layers):
- Layer 1: CVD from OHLCV (50% weight)
- Layer 2: Taker buy/sell ratio (30% weight)
- Layer 3: Notable orders >$5K (20% weight)

Each layer provides:
- score: [-1, +1] directional signal
- Additional metrics for UI display
